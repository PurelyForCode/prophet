GUIDE

Domain folder =  contains all the core classes
Infra folder = contains db, api, and interface implementations
Application Folder = contains the features, uses the classes from domain and infra
Web Folder = Routers

Terminology Guide
1. Router - basic routers
2. Usecase - Represents a single feature
3. Entity - Something with identity
4. Value Object - Represents a value (no identity)
5. DAO - Data access object, talks to the database
6. QueryDAO - Data access object but only does reads
7. Repository - Repository is used in application layer to access database


Important Classes
1. UnitOfWork - Handles transactions
2. DomainEventBus - Handles domain events
